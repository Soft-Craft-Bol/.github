# Soft-Craft Bol 🚀

**Soft-Craft Bol** es una empresa dedicada al desarrollo de soluciones tecnológicas modernas para empresas y emprendimientos.  
Nuestro objetivo es impulsar la transformación digital mediante software eficiente, escalable y adaptado a las necesidades de cada cliente.

🌎 Bolivia  
💻 Desarrollo de Software | Web | APIs | Sistemas Empresariales

## 🚀 Sobre Nosotros

En **Soft-Craft Bol** desarrollamos soluciones digitales enfocadas en mejorar procesos empresariales, automatizar tareas y optimizar la gestión de información.

Trabajamos con tecnologías modernas para construir sistemas:

- Escalables
- Seguros
- Mantenibles
- Orientados a resultados

Nuestro enfoque combina **ingeniería de software, experiencia en desarrollo web y arquitectura de sistemas**.

## 📦 Proyectos Destacados

Estos son algunos de los proyectos desarrollados dentro de nuestra organización:

### 🔎 Motor-Busqueda
Sistema de búsqueda semántica que permite recuperar información basada en el significado contextual de los datos, y no solo coincidencias literales.

Tecnologías:
- Handlebars
- JavaScript

---

### 💳 api-facturacion
API para sistemas de facturación electrónica y gestión de ventas.

Características:
- Gestión de clientes
- Facturación
- Control de pagos
- Integración con sistemas externos

---

### 🌐 web-softcraft
Sitio web institucional de Soft-Craft Bol.

Tecnologías:
- JavaScript
- HTML
- CSS

---

### 🛒 panaderia
Sistema web desarrollado con **React + Vite** para la gestión de ventas y productos de una panadería.

- Control de productos
- Gestión de ventas
- Interfaz moderna y rápida

---

### 💊 Farmacia
